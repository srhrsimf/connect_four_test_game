# Игра "4 в ряд" (Connect Four)

Интерактивная веб-игра "4 в ряд", разработанная с использованием **React** и **TypeScript**. Проект включает в себя полную игровую логику, визуализацию хода, сохранение состояния и инструмент для валидации истории игры.

## Технологии

* **Фреймворк:** React (Vite)
* **Язык:** TypeScript
* **Стили:** CSS3
* **Версионный контроль:** Git / GitHub


❗️❗️❗️❗️❗️❗️ Как запустить проект ❗️❗️❗️❗️❗️❗️

Для запуска проекта локально выполните следующие команды в терминале:

1.  **Клонировать репозиторий:**
    ```bash
    git clone [https://github.com/srhrsimf/connect_four_test_game.git](https://github.com/srhrsimf/connect_four_test_game.git)
    cd connect_four_test_game
    ```

2.  **Установить зависимости:**
    ```bash
    npm install
    ```

3.  **Запустить приложение:**
    ```bash
    npm run dev
    ```
    (Приложение будет доступно по адресу `http://localhost:5173` или аналогичному порту).


## Выполненные требования и особенности

### 1. Структурное разделение кода

* **`src/utils/gameLogic.ts`**: Содержит всю чистую логику игры: функции для хода (`dropPiece`), проверки победы (`checkWin`), проверки ничьей и инициализации доски.
* **`src/App.tsx`**: Отвечает исключительно за рендеринг, обработку событий UI и управление состоянием.

### 2. Валидатор истории ходов

* **`src/utils/validator.ts`**: Реализована функция `validateGameHistory`, которая принимает массив ходов и **полностью воспроизводит игру**.
* При завершении игры в модальном окне доступна кнопка **"Показать историю ходов"**, которая выводит отчет валидатора.

### 3. Улучшения

* **Анимация падения фишки:** При каждом ходе фишка плавно анимируется, падая с верхней части колонки в свою ячейку (с эффектом легкого отскока).
* **Сохранение/Загрузка состояния:** Состояние игры (включая доску, текущий ход и историю) автоматически сохраняется в `localStorage` и восстанавливается при перезагрузке страницы.
* **Визуализация победы:** Четыре фишки, составляющие победную линию, подсвечиваются яркой зеленой обводкой.
* **Интерактивность:** Стрелки выбора колонки подсвечиваются цветом текущего игрока и анимируются при наведении.
